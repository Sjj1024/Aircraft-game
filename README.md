# 飞机大战小游戏

使用的是pygame开发出来的，具有音效，爆炸，发射子弹等效果。
玩家飞机移动控制使用的是鼠标移动，点击鼠标左键发射子弹。
增加了敌军发射子弹的效果，但是还没有做敌军子弹和玩家飞机发生碰撞的效果，新手可以尝试添加此功能，
